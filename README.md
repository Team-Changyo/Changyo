# OverView
우리 삶의 다양한 곳에는 '보증금 시스템'이 존재합니다. 스키장에서의 장비대여 보증금, 펜션을 빌릴 때의 숙소 보증금 등이 그 예시입니다.

'챙겨요'는 이러한 보증금을 오직 QR 스캔과 한번의 터치로만 간편하게 송금하고, 반환할 수 있도록 도와줍니다.

> 보증금도 신한으로, 챙겨요!
